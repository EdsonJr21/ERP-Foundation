# ERP-Foundation

Projeto desenvolvido para estudo de **C#**, **Entity Framework Core**, **MySQL** e **arquitetura de software**, evoluindo gradualmente para um sistema **ERP** com **API REST**, autenticação e testes automatizados.

## Objetivos

* Aprimorar conhecimentos em C#
* Aplicar Programação Orientada a Objetos (POO)
* Implementar padrões de arquitetura em camadas
* Trabalhar com Entity Framework Core e MySQL
* Desenvolver APIs REST com ASP.NET Core
* Aplicar autenticação e autorização
* Utilizar testes automatizados
* Simular funcionalidades presentes em sistemas ERP reais

## Tecnologias

* C#
* .NET
* Entity Framework Core
* MySQL
* LINQ
* Git
* GitHub

## Funcionalidades

- Cadastro de Produtos
- Atualização de Produtos
- Remoção de Produtos
- Consulta de Produtos
- Persistência de dados com Entity Framework Core
- Operações assíncronas com Async/Await
- Consultas utilizando LINQ
